Sobre Nosotros

Hola, nosotros somos el grupo 2, formado por: Alejandro Arbós, Linyi Zhang y Lluis Sans. Somos estudiantes del ITB (Institut Tecnològic de Barcelona) de Administración de Sistemas Informáticos en Red.

Objetivos Como Grupo

Somos un grupo con el interés de desarrollar una página web desde 0, y solo con la ayuda necesaria de inteligencia artifical. Además, queremos crear una página web que no solo tenga información y proyectos, sino que también sea visualmente estética para el agrado del usuario (User Experience)

Objetivos del Proyecto

El desarrollamiento de esta página web es para un trabajo de la asignatura de "Llenguatge de Marques, Sostenibilitat i Digitalització", la cual en esta tarea tiene la finalidad de poner nuestros conocimientos a prueba sobre HTML, CSS i JavaScript. Deberemos de crear una web que destaque especialmente dos de nuestros proyectos en la página principa, crear un índice con otros trabajos/proyectos y un formulario de contacto.
